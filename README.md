# VpnBot

Telegram-бот для управления VPN-пользователями и подписками через Marzban.

Основная идея: у пользователя в чате одно UI-сообщение (панель), которое редактируется кнопками и событиями (напоминания, оплата, статус доступа).

---

## Что умеет бот

- Управление пользователями в Marzban: выдача/удаление/поиск.
- Проверка подписки и выдача конфигов.
- Напоминания об оплате за 2 дня, за 1 день и в день оплаты.
- Авто-ограничение доступа после просрочки оплаты.
- Поток оплаты через скриншот:
  - пользователь отправляет чек;
  - админу приходит карточка с кнопками подтверждения;
  - при подтверждении срок продлевается на месяц;
  - подписка в Marzban снова включается.
- Отчётность по подтверждённым оплатам за месяц.
- Нормализация дат оплат для миграции со старого формата.

## Команды

### Пользователь

- `/start` — открыть главный UI-экран.
- `/help` — показать справку.
- `/info` — инструкции для телефона/ПК/ТВ.
- `/subscribe` — данные подписки + `Оплачен до`.
- `/ping` — проверка доступности RU-сервера.

Основные кнопки панели:
- `🔑 Запросить ключ`
- `📋 Моя подписка`
- `💳 Оплата` (для платных тарифов)
- `📣 Объявления`

### Администратор

Пользователи:
- `/adduser <username>`
- `/deleteuser <username>`
- `/users [username]`
- `/block <id>`
- `/unblock <id>`
- `/blocked`
- `/activity`
- `/count`

Подписки и платежи:
- `/setprice <username> <amount>`
- `/setdate <username> <YYYY-MM-DD>`
- `/setfree <username> <true|false>`
- `/compensation <days>`
- `/override <old YYYY-MM-DD> <new YYYY-MM-DD>`
- `/normalizepay` — привести даты оплаты к текущему `YYYY-MM` (с защитой от двойной оплаты).
- `/paystats [YYYY-MM]` — отчёт по подтверждённым оплатам (кнопка `✅ Подтвердить`).

Служебное:
- `/say` — рассылка объявлений.
- `/poll_result <id>`
- `/poll_list`
- `/status` — серверная нагрузка.

## Оплата и биллинг (текущая логика)

- Дата оплаты хранится как полная дата (`payment_date`).
- Напоминания отправляются по календарным дням:
  - `-2 дня`, `-1 день`, `0 день`.
- Если дата оплаты прошла — доступ в боте ограничивается до оплаты.
- При просрочке подписка в Marzban переводится в `disabled`.
- При подтверждении оплаты админом:
  - срок сдвигается на 1 месяц;
  - статус в Marzban возвращается в `active`;
  - событие попадает в журнал отчётности.

## Установка и запуск

1. Клонировать репозиторий:
```bash
git clone git@github.com:Kovshik387/vpnbot.git
cd vpnbot
```

2. Установить зависимости:
```bash
go mod tidy
```

3. Настроить `.env`.

4. Запустить:
```bash
go run ./cmd/api
```

### Пример `.env`

```env
BOT_TOKEN=string
ADMIN_ID=int64
USERNAME_API=string
PASSWORD_API=string
MARZBAN_URL=http://127.0.0.1:8000
RUSSIAN_URL=http://127.0.0.1:8000
```
